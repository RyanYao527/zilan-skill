# 手动上传步骤

## 步骤 1：在 GitHub 新建仓库

1. 登录 GitHub：https://github.com
2. 点击右上角 **+** → **New repository**
3. 填写：
   - **Repository name**：`zilan-agent`
   - **Description**：`孜澜 - 独立佛学修行者 Agent · 阿含检索、因明推理、摄类学分析与日常修学支持`
   - **可见性**：**Public**（开源）
   - ✅ **勾选** "Add a README file"（初始化用）
4. 点击 **Create repository**

---

## 步骤 2：克隆新仓库到本地

在终端执行（把 `YOUR_USERNAME` 换成你的 GitHub 用户名）：

```bash
git clone https://github.com/YOUR_USERNAME/zilan-agent.git
cd zilan-agent
```

---

## 步骤 3：拷贝文件进去

把临时打包目录的内容全部拷贝到仓库里：

```bash
# Windows PowerShell
Copy-Item -Path "<LOCAL_PACKAGE_DIR>\*" `
           -Destination "." -Recurse -Force

# 或 Mac/Linux
# cp -r /tmp/zilan-agent-pack/* .
```

---

## 步骤 4：提交并推送

```bash
git add .
git commit -m "feat: initial 孜澜数字人 skill v1.1"
git branch -M main
git push -u origin main
```

---

## 完成后

你的仓库地址会是：
```
https://github.com/YOUR_USERNAME/zilan-agent
```

---

## 安装到本地 Claude Code

克隆仓库后，把 skill 目录拷贝进去：

```bash
# 克隆你的仓库
git clone https://github.com/YOUR_USERNAME/zilan-agent.git

# 拷贝到 Claude Code skills 目录
# 路径中的中文目录名保持不变
cp -r zilan-agent ~/.claude/skills/

# 验证
Get-ChildItem ~/.claude/skills/zilan-agent
```

---

## 如果你想之后同步更新

每次 skill 有新内容时：
```bash
git add .
git commit -m "feat: 更新内容描述"
git push
```
