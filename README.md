## 是什么

这是一个为 AI 大语言模型提供的**用户深度背景对齐skill**。

它的核心功能：当你在任意 AI 对话中唤醒"孜澜"、"姚磊"、"优婆塞"或"数字人佛学"时，AI 将加载你的完整认知画像、修学阶段、摄类学工具箱、《大藏经》映射模型、日常修行状态参数，并以符合你认知框架的方式进行佛学探讨和日常修行支持。
